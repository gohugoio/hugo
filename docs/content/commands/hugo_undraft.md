## hugo undraft

Undraft changes the content's draft status from 'True' to 'False'

### Synopsis


Undraft changes the content's draft status from 'True' to 'False'
and updates the date to the current date and time.
If the content's draft status is 'False', nothing is done.

```
hugo undraft path/to/content
```
